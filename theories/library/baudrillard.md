# ジャン・ボードリヤール（Jean Baudrillard）

### 1. 概要
* **主要著作:** 『シミュラークルとシミュレーション（Simulacres et Simulation）』（1981年）
* **基本概念:** オリジナルのないコピー、あるいは実在よりもリアルな「擬似的な現実」が世界を覆い尽くす状態を論じた。
* **後世への影響:** ポストモダン哲学の代表的論考であり、映画『マトリックス』の着想源となったことでも知られる。

### 2. 核心的記述：シミュラークルの4段階
ボードリヤールは、イメージ（像）が現実と切り離されていく過程を4つの段階で説明した。

1. **第一段階:** 現実の深い反映（良きイメージ）。
2. **第二段階:** 現実を覆い隠し、変質させる（悪きイメージ）。
3. **第三段階:** 現実が不在であることを隠蔽する。
4. **第四段階:** 現実とは全く無関係になり、それ自身の純粋な「シミュラークル（擬似像）」となる。

> **原文（フランス語）**
> "Le simulacre n'est jamais ce qui cache la vérité — c'est la vérité qui cache qu'il n'y en a pas. Le simulacre est vrai."
> 
> **日本語訳**
> 「シミュラークルは真実を隠すものではない。真実こそが、そこには何もないということを隠しているのである。シミュラークルは真実なのだ。」

### 3. ハイパーリアル（超現実）
現実のモデルが、現実そのものよりもリアルに感じられる状態についての洞察。

* **オリジナルの消失:** デジタルデータのように「無限に複製可能なもの」においては、もはや参照すべき「オリジナル」は重要ではなくなり、コピー同士が相互に参照し合うようになる。
* **ハイパーリアリティ（Hyperreality）:** 現実よりも「現実らしい」記号やイメージによって構築された世界。それは現実の欠陥を修正し、完璧に制御された「超現実」として現れる。

### 4. 本書からの直接的な示唆
* **地図が領土に先立つ:** かつては領土（現実）が地図（イメージ）に先んじていたが、現代では地図（シミュレーション）が先にあり、現実がそれに合わせるようになる。
* **ノスタルジーの発生:** 本物の現実が失われるほど、人々は「本物らしさ（オーセンティシティ）」を過剰に求めるようになり、失われた現実をシミュレーションの中で再現しようとする。
* **シミュレーションの完結性:** システムが完璧にシミュレーションを行うとき、そこには外部（客観的な現実）との接点がなくなり、シミュレーション内部だけで意味が完結する。
