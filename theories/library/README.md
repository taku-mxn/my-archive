# 📚 Theories Library (知のライブラリ)

このディレクトリは、「空間コンテンツ学」を構築するための基礎理論や先行研究をストックする「知のパーツ置き場」です。

## 📍 このライブラリの位置づけ

ここにあるドキュメントは、各文献の要約・エッセンスのストックです。
原典も一通り目を通してはいますが、記載内容はAI（Gemini）サマリを転記しています。
現時点（20260412）では以下の状態にあります：
- **原典の未検証:** 執筆者がすべてのAIサマリと原典との整合を直接目視し、精読・確認した内容ではありません。
- **ドラフト段階:** あくまで思考を整理するための「仮置き」のサマリです。

## 🛠 今後の運用方針

1. **原典との照合と修正:** 今後、順次原典にあたり、内容の正確性を検証します。必要に応じて、記述の修正や追記を随時行います。
2. **思索への接続:** ここにある理論を前提（モジュール）として引用し、`theories/thoughts` ディレクトリにて独自の考察を展開します。

## 💬 閲覧者の皆様へ（意見・異論の歓迎）

内容の誤り、解釈の飛躍、あるいは「この文献にはこういう側面もあるのではないか」といった異論や補足があれば、ぜひIssueやPull Requestを通じて提起いただけますと幸いです。
